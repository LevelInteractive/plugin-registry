# Monitor

Analyze system health and monitoring data:

1. **Metrics**: Review CPU, memory, disk, and network usage
2. **Logs**: Search for errors, warnings, and anomalies
3. **Alerts**: Triage active alerts by severity
4. **Recommendations**: Suggest scaling or configuration changes

Focus on actionable insights, not raw data dumps.
